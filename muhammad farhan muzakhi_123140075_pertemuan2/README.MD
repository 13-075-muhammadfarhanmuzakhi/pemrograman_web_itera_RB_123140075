Personal Dashboard: Aplikasi Daftar Tugas Sederhana

Personal Dashboard adalah aplikasi daftar tugas (To-Do List) sederhana berbasis web yang memungkinkan pengguna untuk menambah, mengedit, dan menghapus tugas, serta menampilkan jam real-time. Data tugas disimpan secara lokal menggunakan localStorage.

Fitur Utama

    Daftar Tugas (To-Do List): Tambah, edit, dan hapus tugas.

    Penyimpanan Lokal: Tugas disimpan menggunakan localStorage browser, sehingga data tetap ada meskipun halaman dimuat ulang.

    Jam Real-time: Menampilkan waktu saat ini yang diperbarui setiap detik.

    Desain Responsif & Modern: Menggunakan desain minimalis dengan gradient background dan efek visual yang menarik.

    Struktur Kode Berorientasi Objek: Menggunakan Class (ES6) untuk representasi objek Task.

Teknologi yang Digunakan

    HTML5 (index.html): Struktur dasar dan konten halaman.

    CSS3 (style.css): Penataan gaya, tata letak (layout), dan tampilan visual.

    JavaScript (ES6+) (script.js): Logika aplikasi, manipulasi DOM, manajemen localStorage, dan fungsi jam.
