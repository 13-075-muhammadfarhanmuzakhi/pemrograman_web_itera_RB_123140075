Program menampilkan seluruh data mahasiswa dalam bentuk tabel yang berisi:
Nama
NIM
Nilai UTS
Nilai UAS
Nilai Tugas
Nilai Akhir (dihitung otomatis)
Grade berdasarkan nilai akhir

Pengguna dapat menambahkan data mahasiswa dengan menginput:
Nama
NIM
Nilai UTS
Nilai UAS
Nilai Tugas
Data yang dimasukkan akan langsung tersimpan ke dalam list dan bisa ditampilkan kembali.

Menampilkan Mahasiswa dengan Nilai Tertinggi dan Terendah
Program akan:
Menghitung nilai akhir seluruh mahasiswa
Menentukan siapa yang mendapatkan nilai tertinggi dan terendah
Hasil ditampilkan dengan nama mahasiswa dan nilai akhirnya.

Filter Berdasarkan Grade
Pengguna dapat memasukkan grade (A/B/C/D/E), lalu program akan menampilkan mahasiswa yang memiliki grade tersebut.

Struktur Program
Program ini terdiri dari beberapa bagian utama:
List data mahasiswa awal
Fungsi perhitungan nilai dan grade
Fungsi tampilan data
Fungsi manipulasi data (tambah, filter, perbandingan nilai)
Menu interaktif untuk menjalankan fitur

Program ini dibuat untuk membantu proses pengelolaan nilai mahasiswa secara sederhana dan interaktif. Cocok digunakan untuk pembelajaran pemrograman dasar Python, khususnya materi:
List
Dictionary
Fungsi
Perulangan
Percabangan
Input/Output terminal
