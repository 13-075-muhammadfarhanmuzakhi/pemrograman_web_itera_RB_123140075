Aplikasi Manajemen Tugas Mahasiswa adalah aplikasi berbasis web yang digunakan untuk mencatat, mengelola, dan memantau daftar tugas kuliah. Aplikasi ini berjalan sepenuhnya di sisi klien dan menyimpan data menggunakan LocalStorage, sehingga tugas yang ditambahkan tetap tersimpan meskipun browser ditutup.

Aplikasi ini dirancang dengan antarmuka yang modern, responsif, dan mudah digunakan oleh mahasiswa.

Fitur Utama
Manajemen Tugas

Menambah tugas baru (nama tugas, mata kuliah, deadline).

Mengedit tugas yang sudah dibuat.

Menghapus tugas.

Menandai tugas sebagai selesai atau belum selesai.

Navigasi & Filter

Fitur pencarian tugas secara realtime.

Filter berdasarkan status (Semua, Selesai, Belum Selesai).

Filter berdasarkan mata kuliah.

Statistik

Menampilkan jumlah total tugas.

Menghitung jumlah tugas yang belum selesai.

Penyimpanan

Semua data disimpan otomatis ke LocalStorage.

Tidak memerlukan backend atau database eksternal.
