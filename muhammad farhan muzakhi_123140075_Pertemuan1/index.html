README — Aplikasi Manajemen Tugas

🧩 Deskripsi Singkat

Aplikasi ini dibuat untuk membantu mahasiswa dalam mengelola daftar tugas kuliah. Dilengkapi dengan fitur CRUD (Create, Read, Update, Delete), pencarian, filter, dan penyimpanan otomatis menggunakan localStorage.

⚙️ Fitur Utama

  • Tambah, edit, dan hapus tugas dengan mudah.
  • Tandai tugas sebagai selesai atau belum selesai.
  • Filter tugas berdasarkan status atau mata kuliah.
  • Pencarian cepat berdasarkan nama tugas.
  • Data tersimpan otomatis di localStorage.
  • Statistik total tugas dan tugas belum selesai.

🖥️ Teknologi yang Digunakan

  • HTML5 — struktur halaman.
  • CSS3 — tampilan modern dan responsif.
  • JavaScript (ES6) — logika dan interaksi dinamis.
  • LocalStorage — penyimpanan data lokal.
